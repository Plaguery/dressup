# Description

A simple dress-up game! i drew all the sprites myself :D

# Development

This project is built using just vanilla js, html, and css. this project is pretty js reliant haha.

all the various items you can wear are represented using OOP. they're stored in arrays, and from there everything is controlled using event listeners. i used .classList to style currently selected buttons!

this project is responsive, so it should look relatively normal in all screen sizes! it's definitely built with pc dimensions in mind though.
